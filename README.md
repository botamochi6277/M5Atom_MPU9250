# M5Atom MPU9250

[M5Atom Lite](https://docs.m5stack.com/#/en/core/atom_lite) connects MPU9250 with software I2C.

| M5Atom Lite |     | MPU9250 |
| :---------: | :-: | :-----: |
|     3V3     | --  |   VCC   |
|     GND     | --  |   GND   |
|     G25     | --  |   SDA   |
|     G21     | --  |   SCL   |
